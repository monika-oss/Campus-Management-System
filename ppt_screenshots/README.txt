Intha folder la unnoda actual website screenshots save pannu.
Exactly intha filenames use pannu (PNG format):

  01_login.png              -> Login page screenshot
  02_admin_dashboard.png    -> Admin Dashboard page
  03_admin_students.png     -> Admin > Students Management page
  04_admin_faculty.png      -> Admin > Faculty Management page
  05_admin_leave.png        -> Admin > Leave Management page
  06_admin_results.png      -> Admin > Results page
  07_admin_announce.png     -> Admin > Announcements page
  08_faculty_dashboard.png  -> Faculty Dashboard page
  09_faculty_attendance.png -> Faculty > Mark Attendance page
  10_faculty_leave.png      -> Faculty > Leave Approvals page
  11_student_dashboard.png  -> Student Dashboard page
  12_student_leaves.png     -> Student > My Leaves page
  13_student_results.png    -> Student > Academic Grades page

After saving all screenshots with these exact names,
run the PPT script:

  python make_ppt.py

The PPT will be saved to Desktop as Smart_Campus_Presentation.pptx
